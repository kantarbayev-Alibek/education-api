# Education API

REST API на Spring Boot для управления институциями.

## Endpoints
- GET /institutions — получить список всех институций
- POST /institutions — добавить новую институцию
- GET /institutions/{id} — получить институцию по ID
- DELETE /institutions/{id} — удалить институцию

## Запуск
- mvn spring-boot:run
- или через IntelliJ: запустить класс EducationApiApplication
